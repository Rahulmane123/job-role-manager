# Job Role Management System

## 📌 Project Overview
This is a full-stack MERN application to manage job roles. It allows users to create, view, update, delete, search, and sort job roles efficiently.

---

## 🚀 Features
- Create, Read, Update, Delete (CRUD) Job Roles
- Search by Job Title and Department
- Sort by Created Date
- Global State Management using Redux Toolkit
- Loading & Error State Handling using Redux
- Responsive user interface

---

## 🛠️ Tech Stack
- Frontend: React (Hooks)
- State Management: Redux Toolkit
- Backend: Node.js, Express.js
- Database: MongoDB
- UI: Bootstrap
